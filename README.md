# MailForge Pro v5.0

> **10 professional email marketing tools. 100% local — nothing ever leaves your browser.**

🔗 **Live:** https://ahmedsiyam444.github.io/MailForge-Pro/

## ✨ Tools

| # | Tool | Description |
|---|------|-------------|
| 1 | **LeadSeed** | Seed test emails into CSV at fixed or random intervals |
| 2 | **List Splitter** | Split email lists into chunks, download as ZIP |
| 3 | **Variation Generator** | Generate subjects & bodies with spam-trigger checker |
| 4 | **Extractor & Cleaner** | Extract emails from raw text, remove bounces |
| 5 | **Deduplicator** | Remove duplicates + Gmail normalisation + cross-list compare |
| 6 | **Bulk Tools** | Find & Replace (regex), sort, case transforms, delimiter changer |
| 7 | **Randomizer** | Fisher-Yates shuffle with seeded PRNG + sample-N mode |
| 8 | **Image Renamer** | Batch rename up to 2000 images, download as ZIP |
| 9 | **Column Converter** | Columns → Delimited / JSON / SQL / Markdown / HTML |
| 10 | **Email Verifier** ⭐ NEW | Syntax, disposable-domain, role-account scoring — offline |

## 🚀 v5.0 Highlights

- Dark / Light / System theme (persisted)
- Command Palette (Ctrl+K or /) — fuzzy search all tools
- Keyboard shortcuts 1–9, 0 to jump to any tool
- Spam trigger checker in Variation Generator
- Seeded PRNG in Randomizer (reproducible shuffles)
- Cross-list dedup (A∩B, A−B, B−A, A∪B)
- Gmail normalisation (strips dots & +tags)
- Find & Replace with regex in Bulk Tools
- JSON / SQL / Markdown / HTML output in Column Converter
- Offline email verifier (200+ disposable domains bundled)
- PWA — installable, offline after first load
- Zero analytics, zero network requests after load
